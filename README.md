- Python 3.10+  
- FastAPI  
- Node.js >= 18  
- Angular CLI  


## Desarrollo  
### Backend  
```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```

La API se expone en http://localhost:8000  

### Frontend  
```bash
cd frontend/finanzas-app
npm install
ng serve
```

Acceder en http://localhost:4200
